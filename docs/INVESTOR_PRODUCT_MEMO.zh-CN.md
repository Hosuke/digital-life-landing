# 数字生命项目投资评估备忘录（产品视角）

## 1. 投资人最关心的问题
- 这是“可持续业务”还是“一次性 Demo”？
- 用户愿不愿意付费，付费后是否持续留存？
- 成本结构能否随规模下降，形成健康单位经济模型？

## 2. 我们的答案（可验证）
- 业务形态：IM 内长期陪伴服务，天然高频复访，不是单次交付。
- 变现逻辑：Trial 低门槛获客 + 情感 Aha 激活 + 阈值升级转化。
- 单位经济：通过模型分级、库存复用、阈值控制管理 COGS。

## 3. 目标用户与市场切入
- 核心早期用户：有情感纪念需求、愿意尝试 AI 交互的家庭用户。
- 切入场景：纪念陪伴、家庭情绪支持、成长记录对话。
- 为什么现在：多模态模型成熟 + IM 生态稳定 + 生成成本下降。

## 4. 产品形态
- 前台：科幻叙事 landing（建立仪式感）
- 中台：UID 驱动控制面（追踪全链路）
- 后台：丫丫运行时（人格/内容/媒体生成）
- 交付界面：Telegram/WhatsApp

## 5. 增长与转化飞轮
1. 内容触达获客 -> 进入 landing
2. UID 深链接激活 -> 进入 IM
3. 素材首采 + 初始化 -> 触发 Aha
4. 限流与风险提示 -> 触发付费升级
5. 订阅与主题包 -> 提升 ARPU 与留存
6. 口碑传播 -> 反哺获客

## 6. 收入模型
- 主收入：订阅（月/季/年）
- 增值收入：多媒体包、纪念日包、家庭成员扩展包
- 中长期：B2B 白标授权和行业解决方案

## 7. 成本结构
- 可变成本：模型调用（文本/图像/视频）、BSP 消息费用
- 固定成本：工程团队、运营团队、基础设施保底
- 关键控制点：
  - Trial 阶段媒体生成上限
  - 模型路由分层（低价模型优先，高价模型兜底）
  - 素材库存复用与异步任务队列

## 8. 关键里程碑（投资决策可观测）
- M1：UID 绑定率 >= 60%，素材完成率 >= 40%
- M2：Aha 完成率 >= 50%，Trial->Paid >= 8-12%
- M3：D30 留存 >= 20%，LTV/CAC > 2
- M4：形成至少 1 条可复制获客渠道（稳定 CAC）

## 9. 风险与对策
- 质量波动：建立回退链路和“可解释失败提示”
- 合规风险：数据最小化、审计留痕、删除策略
- 成本波动：动态限流、模型替换、供应商议价

## 10. 投资判断建议
若在 8-12 周内达到 M2 指标，项目具备继续放大验证的投资价值；
若 M2 长期不达标，应收缩到垂直场景或 B2B 模式，降低 C 端营销消耗。
